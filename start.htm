Welcome to the REFLEX CES Achilles SOM Development Kit Quick Start Page

 

This release of the Achilles SOM allows you to interact with the board in several ways. 

  • Terminal command line interface through a serial connection using the USB cable

  • Graphical desktop interface with a VNC connection using:

    • Ethernet connection (RJ45 port) using IP address issued to the Achilles SOM by DHCP server on your network

    • Ethernet over USB connection (USB OTG port) at IP address 192.168.10.1

  • Connect to webpage hosted by a webserver running on the Achilles SOM using:

    • Ethernet connection (RJ45 port) using IP address issued to the Achilles SOM by DHCP server on your network

    • Ethernet over USB connection (USB OTG port) at IP address 192.168.10.1

 

To use the Ethernet over USB connection, you will first need to install some drivers.  For a Windows PC, browse to the Drivers folder on the SD Card and install the drivers appropriate for your OS:

 

  • ACHILLES_D64 (for 64-bit OS)

  • ACHILLES_DRV (for 32-bit OS)
